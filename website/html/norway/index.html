Knowledge Graph » Norway

About

The Norwegian Knowledge Graph (KG) is a subset of the full TBFY Knowledge Graph that covers Norwegian procurement data.

On February 2021 the Norwegian KG consisted of more than 2.57 million triples and contained information about:

  • more than 43 000 tenders,
  • more than 10 000 awards, and
  • more than 3 000 companies (for reconciled suppliers in awards).

Usage

Users

The Knowledge Graph (KG) is available as Open Data. The data is available online and can be explored through several services. We also provide monthly data dumps of the KG in N-Triples format, as well as the source data files (from OpenOpps and OpenCorporates) in JSON format that is used to create the KG.

Service URL Description
TBFY Platform & Online Tools http://platform.tbfy.eu An overview of the Knowledge Graph based TBFY Platform & Online Tools.
KG SPARQL endpoint http://54.76.151.143:3030/tbfy/query SPARQL endpoint that allows users to query the Knowledge Graph using SPARQL. We are using Apache Jena Fuseki as the SPARQL server and Apache Jena TDB as the underlying database.

Example:
  • SPARQL query that lists 5 tenders
KG API http://data.tbfy.eu/kg-api REST API that allows you to query the Knowledge Graph. See https://github.com/TBFY/knowledge-graph-API/wiki for more info.

Examples:
  • http://data.tbfy.eu/kg-api/tender?size=5
  • http://data.tbfy.eu/kg-api/award?size=5
  • http://data.tbfy.eu/kg-api/organisation?size=5
KG data dump http://dump.tbfy.eu Monthly data dump of the Knowledge Graph in RDF format containing procurement data and company data (for matching suppliers) for the period January 2019 onwards. In addition to the KG data dump in RDF, we have also made available the source data input for the KG, i.e., procurement release data from OpenOpps and company data (for matching suppliers) from OpenCorporates in JSON format.
YASGUI for KG http://norway-yasgui.tbfy.eu YASGUI (Yet Another SPARQL GUI) is a 3rd party web application that we have set up that allows users to query the Knowledge Graph using SPARQL.

Developers

The Knowledge Graph (KG) is available as Open Source Software. All software components and ontologies developed in the TBFY project to create the Knowledge Graph have been released as open source on GitHub.

Repository URL Description
Knowledge Graph (KG) https://github.com/TBFY/knowledge-graph Repository where all source code, information and documentation for creating, deploying and using the Knowledge Graph are found.
OCDS Ontology https://github.com/TBFY/ocds-ontology Repository for the OCDS ontology.
KG API https://github.com/TBFY/knowledge-graph-API Repository for the core API. It also contains the API documentation and SPARQL queries of the TheyBuyForYou project.
KG API Gateway https://github.com/TBFY/api-gateway Repository for the API Gateway. It provides a flexible abstraction layer and a single entry point that securely manages communication between TBFY clients and online tools via API.
